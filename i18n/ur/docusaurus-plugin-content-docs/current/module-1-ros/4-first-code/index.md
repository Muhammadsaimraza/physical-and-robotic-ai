# باب 4: آپ کا پہلا ROS 2 کوڈ

آپ نے کمانڈ لائن سے ROS 2 کو دریافت کیا ہے۔ اب اپنے نوڈس لکھنے کا وقت ہے۔ اس باب میں، آپ ROS *صارف* سے ROS *ڈویلپر* میں منتقل ہوں گے۔ آپ اپنے کوڈ کو ورک اسپیسز اور پیکیجز میں ترتیب دینا سیکھیں گے، اور آپ اپنے پہلے پائتھن پبلشر اور سبسکرائبر نوڈس لکھیں گے۔

یہ وہ جگہ ہے جہاں کورس تصوراتی سے عملی کی طرف منتقل ہوتا ہے۔ آپ اپنا کوڈ لکھیں گے، مرتب کریں گے، اور چلائیں گے۔ ہم اپنی تدریسی طریقہ کار کی "اے آئی تعاون" کی پرت بھی متعارف کرائیں گے، جو آپ کو اپنے پروگرامنگ پارٹنر کے طور پر جنریٹو اے آئی استعمال کرنے کے لیے پرامپٹس فراہم کرے گی۔

*   **دورانیہ:** 4 اسباق، کل 4 گھنٹے
*   **سطح کی تقسیم:** L2: اے آئی تعاون (آپ اے آئی کی مدد سے کوڈ لکھتے ہیں)
*   **ہارڈ ویئر ٹائر:** ٹائر 1 (کلاؤڈ ROS 2 ماحول یا مقامی انسٹال)
*   **شرائط:** باب 3 کی تکمیل، بنیادی پائتھن کا علم

## سیکھنے کے مقاصد
اس باب کے اختتام تک، آپ اس قابل ہو جائیں گے:

*   ایک ROS 2 ورک اسپیس اور ایک پائتھن پیکیج بنائیں۔
*   clpy کا استعمال کرتے ہوئے ROS 2 نوڈ کے لیے ایک پائتhen کلاس لکھیں۔
*   ایک پبلشر نوڈ نافذ کریں جو ایک ٹاپک پر پیغامات بھیجتا ہے۔
*   ایک سبسکرائبر نوڈ نافذ کریں جو ایک ٹاپک سے پیغامات وصول کرتا ہے اور ان پر کارروائی کرتا ہے۔
*   اپنے پیکیج کو بنانے کے لیے colcon کا استعمال کریں۔
*   اپنے کوڈ کو لکھنے اور اس کی وضاحت میں مدد کے لیے جنریٹو اے آئی کا استعمال کریں۔

## اسباق
*   **سبق 4.1: ورک اسپیسز اور پیکیجز** (60 منٹ)
    *   ROS 2 کی ترقی کے لیے معیاری ڈائرکٹری کا ڈھانچہ سیکھیں اور اپنا پہلا پیکیج بنائیں۔

*   **سبق 4.2: ایک پبلشر لکھنا** (60 منٹ)
    *   ایک پائتھن نوڈ لکھیں جو مسلسل ایک "ہیلو، ورلڈ" پیغام ایک ٹاپک پر شائع کرتا ہے۔

*   **سبق 4.3: ایک سبسکرائبر لکھنا** (60 منٹ)
    *   ایک دوسرا پائتھن نوڈ لکھیں جو ٹاپک کو سبسکرائب کرتا ہے اور موصول ہونے والے پیغامات کو پرنٹ کرتا ہے۔

*   **سبق 4.4: اے آئی کے ساتھ کوشش کریں** (60 منٹ)
    *   فراہم کردہ پرامپٹس کا استعمال کرکے ایک اے آئی اسسٹنٹ سے اپنے کوڈ کی وضاحت کرنے، ترامیم تجویز کرنے، اور ڈیبگ میں مدد کرنے کو کہیں۔
